Sobre o Projeto

Este projeto de site foi desenvolvido com o objetivo de criar uma interface front-end para uma API de conversão de moedas, utilizando o framework Angular. Além de sua funcionalidade prática, o projeto também serve como parte do meu portfólio profissional. A API consumida é de autoria própria e foi construída para consultar dados do Banco Central do Brasil, que disponibiliza as cotações médias diárias de moedas estrangeiras em relação ao Real Brasileiro (BRL).

Tecnologias Utilizadas

  • Angular
  • Tailwindcss
